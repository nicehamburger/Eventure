## ðŸš€ Key Features

### For Entrants (Users)

- **Seamless Access:** No passwords required; users are identified using a device-based identifier.
- **QR Code Registration:** Join event waiting pools instantly by scanning promotional QR codes.
- **Automated Notifications:** Receive real-time updates when selected, waitlisted, or replaced.
- **Invitation Management:** Accept or decline invitations directly in the app. Declined spots are automatically reassigned via redraw.

### For Organizers

- **Event Management:** Create and manage events with registration windows, pricing details, and participant limits.
- **Fair Lottery Selection:** Randomly sample a configurable number of participants from the waiting pool.
- **Optional Geolocation Insights:** View anonymized geographic distribution of entrants to understand community reach.
- **Data Export:** Export confirmed attendee lists to CSV for offline use.

### For Administrators

- **Content Moderation:** Review and remove events, images, or profiles that violate platform guidelines.
- **System Monitoring:** Track notification activity to ensure system transparency and reliability.
